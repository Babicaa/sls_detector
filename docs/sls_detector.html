dacs¶

An instance of DetectorDacs used for accessing the dacs of a single or multi detector.

Examples

d = sls.Detector()

#Set all vrf to 1500
d.dacs.vrf = 1500

#Check vrf
d.dacs.vrf
>> vrf       :  1500,  1500

#Set a single vtr
d.dacs.vtr[0] = 1800

#Set vrf with multiple values
d.dacs.vrf = [3500,3700]
d.dacs.vrf
>> vrf       :  3500,  3700

#read into a variable
var = d.dacs.vrf[:]

#set multiple with multiple values, mostly used for large systems
d.dacs.vcall[0,1] = [3500,3600]
d.dacs.vcall
>> vcall     :  3500,  3600

d.dacs
>>

dynamic_range¶

int – Dynamic range of the detector.

dr max counts comments
4 15  
8 255  
16 4095 12 bit internally
32 4294967295 Autosumming of 12 bit frames
Raises:ValueError – If the dynamic range is not available in the detector
eiger_matrix_reset¶

Matrix reset bit for Eiger.

True : Normal operation, the matrix is reset befor each acq. False : Matrix reset disableld. Used to not reset before reading out analog test pulses.
